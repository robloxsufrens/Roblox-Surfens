## 📌 About

**Roblox Surfens** is a single-file, browser-based endless runner inspired by
Subway Surfers. It's built entirely with vanilla HTML, CSS, and JavaScript —
no build step, no dependencies, no backend required.

### ✨ Features

| Feature | Description |
|---|---|
| 🏃 **Endless runner gameplay** | Dodge obstacles, dash across lanes, collect coins |
| 🎨 **Character select** | Multiple playable characters with live thumbnail previews |
| 🛒 **In-game shop** | Spend coins on skins, boards, and cosmetics |
| 🎯 **Missions & combos** | Mission tracking, combo popups, multipliers |
| 🏆 **Leaderboard** | Local leaderboard with usernames and high scores |
| 📅 **Daily rewards** | Daily login streak bonus system |
| 📱 **Mobile-friendly** | Touch controls, responsive HUD |

## 🚀 Quick Start

### Option 1 — Play Online (GitHub Pages)

Once this repo is pushed to GitHub, enable **GitHub Pages** in
`Settings → Pages → Deploy from branch (main / root)` and the game will be
live at:

```
https://<your-username>.github.io/roblox-surfens/
```

### Option 2 — Run Locally

```bash
git clone https://github.com/<your-username>/roblox-surfens.git
cd roblox-surfens

# Just open it in a browser
open index.html   # macOS
# or double-click index.html on Windows/Linux
```

No build tools, no `npm install` — it's a single static HTML file.
