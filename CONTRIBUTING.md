# Contributing to Roblox Surfens

Thanks for your interest in contributing! 🎉

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your change: `git checkout -b feat/my-feature`
3. **Make your changes** in `index.html` (the game is a single file)
4. **Test locally** by opening `index.html` in a browser
5. **Commit** with a clear message: `git commit -m 'feat: add double-jump powerup'`
6. **Push** your branch: `git push origin feat/my-feature`
7. **Open a Pull Request** describing what you changed and why

## Types of Contributions

| Type | Examples |
|---|---|
| 🐛 Bug fixes | Collision detection, save/load issues, mobile touch bugs |
| ✨ New features | New characters, shop items, missions, power-ups |
| 🎨 Visual polish | Animations, UI/UX improvements, new themes |
| 📝 Documentation | README improvements, code comments |

## Code Style

- Keep the game in a single `index.html` file unless a change genuinely
  requires splitting it up (discuss in an issue first).
- Match the existing naming conventions (camelCase for JS, kebab-case for
  CSS classes/IDs).
- Comment non-obvious game logic.

## Reporting Bugs

Please open an [issue](../../issues/new/choose) with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser/device used

## Code of Conduct

This project follows a [Code of Conduct](CODE_OF_CONDUCT.md). Please be
respectful and constructive.
